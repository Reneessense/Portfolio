Game Controller

Prototyping Tactile Ergonomics

Project Goal

To build a perfect ergonomic fit.

This project focuses on the detailed workflow of creating a customized, ergonomically optimized game controller prototype using advanced digital fabrication techniques. The goal was to produce a tactile object that precisely conforms to the user's hand contours.

Design & Fabrication Process

  1. 3D Scanning: The project began with the 3D scanning of the user's hand in a grip position. This captured the necessary anatomical data for a precise fit.
  2. Data Cleaning and Retopology: The raw scan data was meticulously cleaned and retopologized to create a streamlined, usable mesh suitable for modeling.
  3. Computational Modeling: The refined hand model was used to sculpt the inner contours of the controller shell, ensuring maximum comfort and grip efficiency.
